API keys

This app uses Claude's vision API to read watch photos. This requires a paid Anthropic API key with billing enabled — usage is billed per request. Get a key at console.anthropic.com and add some credit.

When a reference number isn't legible, it's looked up via Gemini with Google Search — Gemini looks at the photo directly and grounds its answer in real Google results. Get a key at aistudio.google.com/apikey, and make sure billing is enabled on the linked Google Cloud project for reliable quota. Free for several thousand lookups/month, then a small per-lookup charge after that. Leave blank to skip lookup (no reference will be shown for unclear photos rather than a guess).

Both keys are stored only in this browser, never sent anywhere except directly to Anthropic's and Google's APIs.
